# AJ — Personal Portfolio

Built with a single prompt. React + Vite + GSAP + Lenis.

## Run locally

```
npm install
npm run dev
```

## Deploy

```
npm run build
```

→ drag `/dist` to Vercel, or push and connect repo.

## Customize

Edit `/src/data.js` to update projects and achievements.
Replace `/public/medal.jpg` with your actual medal image.
